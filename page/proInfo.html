预订须知 README

Q :

如果我选择是外拍的服务，那怎么选择服务时间？

A :

我们摄影室每一项外拍服务有规定的最低工作时间，如果需要额外工作时间，可以在订单备注具体注明，消费金额也相应增加。

Q :

为什么预约需要交预约金？如果没时间去怎么办？

A :

我们设置预约金是为了保证安排时间能够充分的利用；如果当天没空可以选择前一天的24:00前进行订单取消，届时系统返回预约金到您账户。

Q :

预约后,在线上交付的预约金如何续回？

A :

可以在消费金额抵消预约金的金额。

Q :

我想通过自己账户帮别人预约拍照，同时我自己也想预约？

A :

您好，由于本平台仅支持个人预约。如有小伙伴需要 进行预约同产品，需要注册一个账号进行预约或该用户的同个产品订单处于已完成/已取消状态。

Q :

关于现场拍照会提供什么服务？

A :

现场会提供专业化妆服务，让你在镜头下展现出最美好的你；如果您选择证件照服务，我们会提供合适的衣服； 其次，我们还有专业修图师进行后期，因此保证照片的真实和美好。
